# Weather API Testing – Postman

## Project Overview

This project contains a Postman collection used to test weather data retrieval through a REST API.

The collection sends GET requests to retrieve current weather information for a specified city and validates the API response.

## Features

* Weather data retrieval by city name
* API request testing
* Query parameter validation
* JSON response verification
* Current weather information validation

## Endpoint Tested

### Get Weather by City Name

Retrieves current weather data for a specified city, including:

* Temperature
* Weather conditions
* Humidity
* Wind information
* Location details

## Tools Used

* Postman
* REST API
* OpenWeather API
* JSON

This project was created to practice:
